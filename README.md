# DAISY - Daesang AI Sales Intelligence for You

This is an AI-powered sales briefing assistant tool based on the requirements specification.

## Features
- Client information input
- Trend briefing generation using AI
- Sales material creation
- Briefing sheet output

## Setup
1. Install Node.js from https://nodejs.org/
2. Run `npm install` in the project directory
3. Get a Gemini API key from https://aistudio.google.com/app/apikey
4. Copy `.env.example` to `.env` and add your API key:
   ```
   GEMINI_API_KEY=your_actual_api_key_here
   ```
5. Run `npm start` to start the server
6. Open http://localhost:3000 in a browser

## Requirements
- Node.js
- Google AI API key for Gemini 1.5 Pro

## Usage
Follow the 3-step process in the web interface.
